BMC Coding Bootcamp

Hack your coding skills

Who is it for?

Anyone with some coding experience who wants a coding career.

Why do it?

  • Get skills
  • Build a Portfolio
  • Meet great people
    • Developers from industry
    • Talented people developing coding (that's you!)
    • Lecturers

What will be covered?

  1. HTML and CSS
  2. JavaScript
  3. Server Side Scripting
  4. CV and Interview Skills
  5. Visits and meeting with local employers

How will it be taught?

We realise that participants in the this course will have verying levels of experiemce
